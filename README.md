## Instalasi

#### Clone proyek
```http
  https://github.com/HardianSur/TryOut.git
```

#### install depedensi
instal projek dengan composer dan npm
```http
  composer install
  npm install
```

#### generate key
```http
  php artisan generate:key
```

#### migrasi database

```http
  php artisan migrate
```

#### Jalankan Proyek
Jalankan projek dengan perintah artisan terlebih dahulu lalu diikuti dengan npm 
```http
  php artisan serve
  npm run dev
```
